## 📦 Projekto technologijos

- **Frontend**: React + Bootstrap
- **Backend**: Node.js (Express)
- **Duomenų bazė**: MongoDB
- **Autentifikacija**: JWT (su email login)
- **Docker & Docker Compose**: visa infrastruktūra paleidžiama konteineriuose
